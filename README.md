# 💸 FinFlow — Gestor de Finanzas Personal

App web progresiva (PWA) para gestionar tus finanzas personales desde el celular.

## ✨ Funcionalidades

- **Inicio** — Balance total, ingresos vs gastos, tasa de ahorro, movimientos recientes
- **Resumen** — Gráfica de torta por categorías, barras de los últimos 6 meses, historial completo
- **Metas** — Crea objetivos de ahorro con barra de progreso
- **Futuro** — Proyecciones a 1, 3, 6 y 12 meses + recomendaciones personalizadas

## 🚀 Cómo subir a GitHub Pages

1. Crea un repositorio en GitHub (ej: `finflow`)
2. Sube los archivos `index.html` y `manifest.json`
3. Ve a **Settings → Pages → Source: main branch → Save**
4. Tu app estará en: `https://tu-usuario.github.io/finflow`

## 📱 Instalar en el celular

### Android (Chrome)
1. Abre la URL en Chrome
2. Toca el menú (⋮) → "Añadir a pantalla de inicio"
3. ¡Listo! Aparece como app nativa

### iPhone (Safari)
1. Abre la URL en Safari
2. Toca compartir (□↑) → "Añadir a pantalla de inicio"
3. ¡Listo!

## 💾 Datos
Los datos se guardan localmente en el navegador (localStorage). No se envía nada a ningún servidor.
